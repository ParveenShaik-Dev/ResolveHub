# ResolveHub

A role-based enterprise service management platform 
built with ASP.NET Core Web API and Angular.

## Features
- Customer can raise service tickets
- Agent can view and resolve assigned tickets  
- Admin can manage users, agents and all tickets
- JWT Authentication with role-based authorization
- CI/CD pipeline via Azure DevOps

## Tech Stack
- Backend: C#, ASP.NET Core Web API, Entity Framework Core
- Frontend: Angular
- Database: SQL Server
- Authentication: JWT
- Deployment: Azure App Services
- DevOps: Azure DevOps CI/CD

## Architecture
- 3-Tier Architecture
- Repository Pattern
- Service Layer
- Dependency Injection
- SOLID Principles
